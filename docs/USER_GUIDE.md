# User Guide - ASD Screening Tool

## Introduction
Welcome to the ASD Early Screening Tool. This guide will help you complete the screening process.

## Who Should Use This Tool?
- Parents of children aged 18-36 months
- Caregivers observing developmental concerns
- Healthcare professionals for preliminary assessment

## How to Use

### Step 1: Access the Tool
Open your web browser and go to:
```
http://localhost:3000
```

### Step 2: Read the Introduction
- Review the information about the screening
- Note that this is NOT a diagnostic tool
- Understand the disclaimer

### Step 3: Start Screening
Click the "Start Screening" button

### Step 4: Answer Questions
- 10 questions about your child's behavior
- Answer honestly based on observations
- Each question has Yes/No options
- Progress bar shows completion status

### Step 5: Submit
- Review your answers
- Click "Submit Screening"
- Wait for analysis (few seconds)

### Step 6: View Results
Results include:
- **Risk Level**: Low, Medium, or High
- **Confidence Score**: Model's certainty
- **Medical Recommendation**: Next steps

## Understanding Results

### Low Risk (Green)
- Minimal ASD indicators
- Continue routine monitoring
- Regular pediatric checkups

### Medium Risk (Yellow)
- Some indicators present
- Consult pediatrician recommended
- May need further evaluation

### High Risk (Red)
- Multiple indicators detected
- Professional evaluation urgently recommended
- Contact specialist immediately

## Important Notes

⚠️ **This tool does NOT diagnose autism**
- Screening tool only
- Not a replacement for professional diagnosis
- Always consult healthcare providers

## Privacy
- No data is stored
- Answers are not saved
- Each session is independent

## Support
For technical issues or questions, contact your system administrator.
